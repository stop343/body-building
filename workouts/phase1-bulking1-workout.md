## 📈 Progressive Overload — แผน 16 สัปดาห์

| สัปดาห์ | เป้าหมาย |
|---|---|
| 1–3 | ปรับตัว เน้น Form และจังหวะการฝึก |
| 4 | **Deload** — ลด Volume 40%, คง Intensity |
| 5–7 | เพิ่มน้ำหนัก 2.5–5 kg ใน Compound |
| 8 | **Deload** |
| 9–11 | เพิ่ม Rep หรือ Set ใน Accessory |
| 12 | **Deload** |
| 13–15 | Volume สูงสุด / PR Attempts |
| 16 | **Deload** — เตรียมเข้า Phase 2 |

> **RPE Target:** 7–8 (ทุก Working Set ควรเหลือ 2–3 reps ใน tank)
