## 🛏️ หลักการพื้นฐานที่สำคัญที่สุด

1. **นอนหลับ 7–9 ชั่วโมง** → GH และ Testosterone หลั่งสูงสุดในช่วงนี้
2. **ดื่มน้ำ 3–4 ลิตร/วัน** → โดยเฉพาะขณะทาน Creatine
3. **Progressive Overload** → เพิ่มน้ำหนักหรือ Rep ทุกสัปดาห์แม้แค่เล็กน้อย
4. **วัดผลทุก 2 สัปดาห์** → ชั่งน้ำหนัก + วัดรอบเอว ปรับแผนตามข้อมูล ไม่ใช่ความรู้สึก
5. **ตรวจเลือดทุก 6 เดือน** → Testosterone, Thyroid, CBC
